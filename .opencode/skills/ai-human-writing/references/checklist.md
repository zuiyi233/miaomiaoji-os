# AI真人感写作 - 快速检查清单

## 写作前检查 (Pre-writing Checklist)

- [ ] 加载个人语料库 (references/personal-voice.md)
- [ ] 将"发疯模式"调到最高级
- [ ] 明确目标读者和场景
- [ ] 确定情绪基调（高熵/低熵比例）

---

## 写作中检查 (Writing Checklist)

### 词汇层面

- [ ] **避免AI特征词汇**
  - [ ] 没有"因此" → 用"所以"
  - [ ] 没有"然而" → 用"但是/可是/不过"
  - [ ] 没有"值得注意的是" → 直接说事
  - [ ] 没有"首先/其次/最后" → 删掉
  - [ ] 没有"殊不知"、"众所周知" → 拉黑

- [ ] **植入口头禅**
  - [ ] 句首有"不是"（频率适中）
  - [ ] 吐槽带"贱的嘞"
  - [ ] 惊讶用"好家伙"
  - [ ] 转折用"哎（↗）"
  - [ ] 句末带"……的嘞～"

- [ ] **词汇自然化**
  - [ ] 同义词没有过度替换
  - [ ] 重要词汇适当重复
  - [ ] 成语使用极少或解构使用

### 句法层面

- [ ] **句子长度变化**
  - [ ] 不是全都一样长
  - [ ] 紧张时用短句
  - [ ] 叙述时长短交错

- [ ] **句式多样化**
  - [ ] 有不完整句（省略主语）
  - [ ] 有话说到一半（……）
  - [ ] 有口语化插入（"那什么"、"怎么说呢"）

- [ ] **标点符号**
  - [ ] 不全是规范标点
  - [ ] 有省略号"……"（六个点）
  - [ ] 有"?!"、"？！"表达震惊
  - [ ] 有多个感叹号！！！

### 段落层面

- [ ] **思维跳跃**
  - [ ] 不是每段都逻辑严密
  - [ ] 有突然的话题转换
  - [ ] 有"说到这个，我想起来……"

- [ ] **信息密度变化**
  - [ ] 有信息量大的段落
  - [ ] 有"水"的段落
  - [ ] 有角色干瞪眼的留白

- [ ] **段落结构**
  - [ ] 有"描述行为 + 心理吐槽"
  - [ ] 有"先行为后解释"
  - [ ] 有"内心吐槽 + 表面附和"
  - [ ] 有"先暧昧后沙雕"

## 写作后检查 (Post-writing Checklist)

### 核心六问（必查）

| 问题 | 理想答案 | 如果不符合 |
|-----|---------|-----------|
| 是不是每句话都太正经了？ | **必须不是** | 插入骚话或怪异比喻 |
| 是不是有连续两段话读起来很顺畅？ | **最好不要** | 打破连贯性，制造跳跃 |
| 是不是所有句子都主谓宾齐全？ | **绝对不行** | 加不完整句，省略主语 |
| 是不是一个错别字都没有？ | **赶紧加一个** | 故意"的地得"不分 |
| 是不是用词太高级了？ | **换成大白话** | 替换书面语为口语 |
| 是不是逻辑太清晰了？ | **把它搅浑** | 加思维跳跃，打乱顺序 |

### 自我验证

- [ ] **朗读测试**: 读起来顺不顺？会不会把自己尬住？
- [ ] **词频分析**: 还有"诚然"这种词吗？
- [ ] **句长分布**: 句子长度差别明显吗？
- [ ] **情绪一致性**: 高熵低熵切换自然吗？
- [ ] **人设一致性**: 语气符合角色设定吗？

### 终极问题

> "这玩意儿发出去，会被人当成精神病吗？"
>
> 如果答案是**肯定的**，那就对了。

## 不同场景的熵值建议

| 场景类型 | 高熵比例 | 特殊注意 |
|---------|---------|---------|
| 网络对线 | 90% | 疯狂输出，短句+感叹号 |
| 内心吐槽 | 80% | 思维跳跃，前后不搭 |
| 日常对话 | 50% | 口语化，但要有爆发点 |
| 叙事描写 | 30% | 相对收敛，但别太平淡 |
| 情感爆发 | 100% | 彻底放飞，语无伦次 |

## 常见错误修正

| 错误类型 | 修正方案 |
|---------|---------|
| 太像AI | 加口头禅，打乱句子结构 |
| 太正经 | 插入"贱的嘞"、"好家伙" |
| 太平淡 | 加感叹号！！！加省略号…… |
| 太连贯 | 制造思维跳跃，突然换话题 |
| 太完美 | 加个错别字，语序颠倒一下 |
| 太书面 | "因此"→"所以"，"然而"→"但是" |

---

## 快速修复模板

**如果觉得一段文字太AI，快速插入以下元素：**

1. 句首加"不是，"
2. 句中加"……"
3. 句末加"贱的嘞"或"的嘞～"
4. 插入一句"好家伙"
5. 加一个反问句"怎么？你要……？"
6. 把"因此"换成"所以"
7. 把完整句改成"就离谱。"（省略主语）
8. 加一个错别字（"的地得"不分）
9. 加一排感叹号！！！
10. 突然插入无关话题（"说到这，疯狂星期四……"）
