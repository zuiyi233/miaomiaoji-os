# 个人语料与表达指纹

## 1. 口头禅与常用表达

### 1.1 核心口头禅

| 口头禅 | 使用情境 | 示例 |
|-------|---------|------|
| "无所谓，我会出手" | 面临困境，决定反击 | "无所谓，她会出手" |
| "无所谓，她自会出殡" | 处境糟糕，决定摆烂 | "无所谓，她自会出殡" |
| "……贱的嘞" | 吐槽他人骚操作 | "真是……贱的嘞～" |
| "这不巧了么这不是？" | 发现有利情况 | "这不巧了吗这不是？" |
| "……的嘞～" | 句末语气助词 | "美的嘞～" |
| "你最好有事" | 不耐烦的沟通 | "你最好有事。" |
| "你最好是" | 怀疑、警告 | "你最好是。" |
| "你介似揍嘛呢？" | 震惊/无语，方言 | "你介似揍嘛呢？" |
| "俺能摧眉折腰事权贵，俺能" | 魔改诗句，表"能屈能伸" | "俺能摧眉折腰事权贵，俺能。" |
| "嘿（↗）"/"哎（↗）" | 转折、惊讶预告 | "嘿（↗）"江绮遇差点被他噎死 |
| "好家伙" | 表示惊讶 | "好家伙，怎么涨这么多粉？" |
| "这很难评" | 无语、无奈 | "这很难评，毕竟隔行如隔山" |
| "好好好" | 反讽、敷衍 | "好好好。" |
| "嘶——" | 震惊、意识到不对 | "嘶……这事不对啊" |
| "怎么说呢" | 评价前的铺垫 | "怎么说呢，你们几个的配合……" |
| "你小子" | 亲昵、调侃 | "你小子，姐还治不了你。" |
| "不是" | 句首反驳/疑问 | "不是，哥们你还要杀人啊？" |
| "那什么" | 思考、转换话题 | "那什么，乔爷爷……" |
| "行吧" | 无奈接受 | "行吧。" |

### 1.2 固定句式

**转折/递进句式**:
- "本来只是……没想到……"
- "虽说话糙理不糙，可这话也太糙了点吧？"
- "但是……"

**"不是……"句式**:
- "不是哥们，你还要杀人啊？"
- "不是，你打错了。"

**"……怎么了？"句式**:
- "怎么了？"
- "怎么了？我脸上长花了？"

**时间/场景转换**:
- "下一刻，……"
- "与此同时，……"
- "紧接着，……"
- "话音刚落/话还没说完，……"

**反问句式**:
- "你以为我就是为了那几个臭钱？"
- "我什么时候答应跟你结婚了？"
- "怎么？你要转行去民生频道？"

## 4. 段落结构偏好

### 4.1 典型结构模式

**模式1: 描述行为 + 心理吐槽**
```
看到被自己骗了后又晾了许久的顾寻简，怕被报复的江绮遇下意识就想赶紧进门。
"日记老师冷静！"
兀的，她闻到了手机中传来的一股糊味，睁大眼睛去看。原来是某人烧起来了——
她发誓，这小子绝对是故意的。
```

**模式2: 先行为后解释**
```
在机场骑电动行李箱引人侧目，被质疑后，内心独白解释……
```

**模式3: 内心吐槽 + 表面附和**
```
面对乔老爷子认错人，江绮遇内心OS："这老爷爷说的话每个字分开她都认识……"
但嘴上还是配合地回应："来、来了……"
```

**模式4: 先暧昧后沙雕**
```
祁逾深情告白后，江绮遇的关注点却是"祁氏破产了？！"
```

**模式5: 先抑后扬/先扬后抑**
```
何初阳深情告白，看似感人，江绮遇却回应"致谢？你这明明是污蔑。"
```

### 4.2 节奏特点

- 短句为主，节奏明快
- 长短句结合用于叙事和心理
- 短句对话，节奏快
- 情绪影响段落长度

## 5. 思维模式与表达逻辑

### 5.1 联想跳跃

**从A跳到B**:
- 安慰小孩 → 思考人生
- 被误解 → 自我营销
- 恋友圈规则 → 发疯文学
- 从被骂 → 釜山行
- 绑架 → 工作机会

**脑补狗血剧**:
- "《富豪之女身患绝症……》这样的狗血戏码。"
- "《平民女孩家中突遭变故……》的狗血戏码。"

### 5.2 情绪表达

**精神状态不稳定** → 语言风格剧变
**记仇与恶趣味** → 蔫儿坏
**紧张/心虚** → 嘴硬/死不承认
**醉酒** → 大胆直白，琼瑶剧
**窘迫/心虚** → 倒打一耙
**愤怒** → 阴阳怪气/毒舌

## 6. 常见场景说话方式

### 6.1 安慰场景

**反向安慰/成人式安慰**:
```
"没关系的，天下没有不散的筵席，你习惯习惯就好了。"
"我们只是胖了点，又不是坏人，有什么好难过的呢？"
```

### 6.2 批评/互怼场景

**反应极快，抓住漏洞反击**:
```
"老登嘴这么碎？平时自己在家没少偷吃鸡饲料吧？"
"我建议你买瓶红酒回去洒在油锅里，看看到底是油溅还是你贱。"
```

### 6.3 紧张/尴尬场景

**插科打诨、转移话题**:
```
"我要——不还是算了吧，突然想起来我房间好像着火了……"
```

## 9. 自然化缺陷

### 9.1 口误/谐音梗

- "名声在外，有好有坏，以前是以前，现在是变态" → "现在"
- "打开天窗说脏话" → "说亮话"
- "他不走我吃不下" → "他没消息我吃不下"
- "我跟简子心连心，简子你跟我玩脑筋？"

### 9.2 口语化省略

- 主语省略："（我）看出来了。"
- 连接词省略："他一个放高利贷起家的，（所以）钱来的可不要太容易。"

### 9.3 标点随意性

- 省略号"……"使用非常频繁（六个点）
- 破折号"——"用于解释和声音延长
- 波浪线"～"用于俏皮语气
- "?!"、"？！"频繁使用表震惊

### 9.4 语法不完美

- 语序调整："这事我真没想到"
- 不完整句："要是能早点知道..."
- 重复强调："真的真的很重要"
